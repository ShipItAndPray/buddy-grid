# BuddyGrid

**[Live Demo](https://shipitandpray.github.io/buddy-grid/)**

The complete species encyclopedia for Claude Code `/buddy`. All 18 species, all 5 rarities, 90 unique combinations.

## Features

- **18x5 Grid** — Every species at every rarity tier, 90 cells total
- **Click any cell** — Modal with full stats, radar chart, soul description, drop rate
- **Filter bar** — Filter by species, rarity, or shiny-only toggle
- **Species lore** — Short lore blurb for each of the 18 species
- **Animated sprites** — CSS idle animations (float, bounce, sway) cycle per card
- **Radar chart** — Canvas-drawn stat pentagon for selected species
- **Search** — "Find your buddy" input highlights matching cells
- **Dark theme** — Masonry-style card layout
- **Mobile responsive** — Collapses to 2-column grid on small screens
- **Zero dependencies** — Single `index.html`, no build step
